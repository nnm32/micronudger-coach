    How to remove Micronudger later (two easy ways)

    1) One‑click removal file

    Use the red “Remove from my calendar (.ics)” button above. Open it on your iPhone/Mac/Google Calendar and confirm.

    This file cancels the events you last generated.

    2) Delete the calendar

    1. iPhone/iPad: Calendar → Calendars → uncheck or tap ⓘ → Delete Calendar.
    2. Mac: Calendar → right‑click “Micronudger – Wellness” → Delete or uncheck.
    3. Google Calendar: hover name → ⋮ → Settings → Remove calendar.

    Tip

    When adding, choose New Calendar named “Micronudger – Wellness”. That makes removal a single step later.
